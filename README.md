# She Can Foundation — Website

A simple, responsive single-page website for **She Can Foundation**, an NGO working with girls and young women in underserved communities through education, mentorship, and life-skills programmes.

## 🌱 Overview

This website was built as a community/volunteer task and intentionally kept simple, while including a few extra creative touches.

## ✨ Features

- **Home/Hero Section** – Welcome message with hero image and call-to-action buttons
- **About Section** – Foundation's mission and core programme pillars (Education, Mentorship, Life Skills)
- **Our Work Section** – Real photo of beneficiaries with an impact story/quote
- **Volunteer / Join Us Button** – Prominent CTA to get involved
- **Footer** – Quick links, contact info, and copyright
- **AI Chatbot (mock)** – "Saanvi" chat assistant with pre-set NGO-related Q&A (programmes, volunteering, donations, contact info)
- **Animations** – Scroll-reveal effects, floating hero image, hover interactions
- **Responsive Design** – Works on desktop, tablet, and mobile (includes a mobile nav menu)

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts: [Fraunces](https://fonts.google.com/specimen/Fraunces) & [Inter](https://fonts.google.com/specimen/Inter)

## 🚀 Getting Started

1. Clone or download this repository.
2. Make sure `images/girls.png` is in the same relative path as `index.html`.
3. Open `index.html` in any modern web browser — no build step or server required.

```bash
git clone <your-repo-url>
cd shecan-foundation
open index.html   # or double-click the file
```

## 🎨 Design

- **Colors:** Forest green (`#1F3D2B`), warm sand (`#F6F1E7`), marigold accent (`#E08E45`), cream background (`#FFFBF5`)
- **Theme:** Growth, education, and empowerment — represented through a sprout/growth motif used as a section divider

## 🤖 Chatbot Note

The chatbot widget uses **mock/static data** for demonstration purposes only — it is not connected to a live backend or AI API.
